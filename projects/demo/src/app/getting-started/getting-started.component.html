Getting started

Installation

In order to install the lib, first credentials needs to be added to .npmrc file. Execute the following command in your terminal for getting the credentials:
curl -u"username":"password" https://artifactory.dagility.com/artifactory/api/npm/npm-local/auth/dagility-ui
// Contact administrator for username and password
For installation of this lib, execute the following command:
npm i @dagility-ui/kit // To access the repository, contact the administrator

Styles

For import styles you must add the following line in your stylesheet:
@import "~@dagility-ui/kit/styles/styles";

UiKitModule

To use components from the library, you must import the UiKitModule into your.module.ts. You can do it in a similar way:
{{ "import { UiKitModule } from '@dagility-ui/kit';" }}
{{ "..." }}
@NgModule({{ "{ " }}
imports: [UiKitModule],
{{ " })" }}
export class YourModule

Update UI Kit

For updating UI Kit you need to make changes, then in projects/kit/package.json find next line:
"version": "0.0.1",
And you need to increment version value by 1 like this:
"version": "0.0.2",
Then do commit and push.
Next you need to increment version in application in package.json like this:
{{ '"dependencies": {' }}
...
{{ '"@dagility-ui/kit": "0.0.2",' }}
...
{{ "\}" }}
